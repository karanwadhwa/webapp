## Quickstart

1. Install npm packages

```console
$ npm install
```

2. Create `.env` file from `.env.example` template.
3. Install MySQL and create an empty database as defined in `.env`
4. Run setup script to initialize database tables

```console
$ npm run setup
```

5. Run unit tests

```console
$ npm run test
```

6. Start node server

```console
$ npm run start
```
